# 개요
* 유튜브 생활코딩의 Next.js 13 강의 클론코딩<br />
(https://www.youtube.com/watch?v=9KOaR6QMb9A&list=PLuHgQVnccGMCwxXsQuEoG-JJ7RlwtNdwJ&index=1)
* nextjs 앱 라우팅 방식을 사용해본다.

# 시작 및 실행
1. 시작
    ```bash
    npm install
    ```
2. 실행
    ```bash
    npm run dev
    ```

# json-server
```bash
npx json-server --port 9999 --watch db.json
```
